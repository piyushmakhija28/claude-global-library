# JavaFX IDE Designer Skill

## Overview

Expert skill for designing and implementing complete IDE (Integrated Development Environment) interfaces using JavaFX.

## Skill Contents

| File | Purpose | Size |
|------|---------|------|
| `skill.md` | Main skill file with complete implementation patterns | ~38KB |
| `QUICK-START.md` | Quick reference guide for common use cases | ~12KB |

## Key Capabilities

- Complete IDE layout design (menu bar, tool bar, file tree, code editor, console)
- Syntax highlighting implementation with RichTextFX
- Dark theme CSS styling
- File tree navigation
- Tab management for multiple files
- Status bar implementation
- Keyboard shortcuts integration

## Technologies

- Java 17+
- JavaFX 21+
- RichTextFX (code editor)
- ControlsFX (additional controls)
